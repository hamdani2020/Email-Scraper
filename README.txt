# Email-Scraping	

This is an advanced Email-Scraping with Python Script allows you to easily and quickly scan bulk of websites and collect email addresses from them.

## How to Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes only. When you are satisfy then we can run it on a live system.

### Prerequisites

What things you need to install the software

```
[Python 3.x](https://www.python.org/downloads/) - A Programing languague
```

## Deployment

* Extract the Zip file
* Open urls.txt file 
* Paste all website urls in the urls.txt file(make sure break line after every url)
* Run the Script
* All Emails in the website will be havested.
